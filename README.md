# OUTech presentation
Who we are, what we do, in a [reveal.js](https://github.com/hakimel/reveal.js) presentation

## Quick start
Run it locally :
```
npm install
npm start
```

Then go to [localhost:8000](localhost:8000) ! You should have access to the internet to view full presentation (specifically to [OUTech](https://outech.fr) and [French robotic cup](https://www.coupederobotique.fr) websites)
